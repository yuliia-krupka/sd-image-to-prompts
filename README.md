# Stable Diffusion - Image to Prompts

Kaggle competition: given an image generated by Stable Diffusion, predict the text prompt that produced it.

## Objective

Map images to their source prompts using image-text embedding similarity.

## Data

| File | Description |
|------|-------------|
| `prompts.csv` | Image IDs and their corresponding prompts |
| `images/` | PNG images generated by Stable Diffusion |
| `sample_submission.csv` | Required submission format |

## Approach

Extract image embeddings using CLIP (ViT-H-14, pretrained on LAION-2B),
then rank candidate prompts by cosine similarity against the image embedding.

## Evaluation

Submissions are scored by cosine similarity between predicted and ground-truth prompt embeddings.
